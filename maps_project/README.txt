This project creates a web GUI using HTML, Javascript and Google Maps API 
that does the following:
• Use the Google Maps API to display a map at a specific location 
• Draws graphics on a canvas using transparency (also known as the alpha or opacity level) and a customized cursor icon
• Provides a graphical user interface (GUI) to users by combining the use of Google Maps and HTML5 features by 
managing the events and the z-index levels
• Calculates the distance between two geographical locations.
